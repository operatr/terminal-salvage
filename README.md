# terminal-salvage

Windows Terminal rice inspired by the Arc Raiders aesthetic — neon stripes, dark background, minimal noise.

![preview](preview.png)

## What's included

| File | Description |
|------|-------------|
| `settings.json` | Windows Terminal config — color scheme, font, opacity, background |
| `arc-raiders-bg.png` | Background image (mirrored neon stripe photo) |
| `fastfetch-config.jsonc` | Minimal fastfetch config — OS, Host, CPU, GPU, RAM, Shell, Terminal, Uptime |

## Setup

1. **Font** — Install [Azeret Mono](https://fonts.google.com/specimen/Azeret+Mono) (variable font)
2. **Background** — Copy `arc-raiders-bg.png` to `%USERPROFILE%\Pictures\Terminal\`
3. **Terminal settings** — Merge or replace `%LOCALAPPDATA%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json`
4. **fastfetch** — Copy `fastfetch-config.jsonc` to `%APPDATA%\fastfetch\config.jsonc`

## Color scheme

`Arc Raiders-ish` — dark gunmetal base, amber accent, dusty teal highlights.
